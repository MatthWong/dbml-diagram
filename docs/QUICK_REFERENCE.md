## ğŸ› Troubleshooting

**Table won't drag?**
- Make sure you're not clicking on the collapse button or color picker
- Try clicking on the table header or body area

**Arrows not updating?**
- This should happen automatically - if not, try refreshing the page

**Drag feels sluggish?**
- Check if you have many relationships (50+)
- Try collapsing some tables to reduce visual complexity
